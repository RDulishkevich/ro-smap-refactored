# Правила RO.SMap (сообщество и публикация)

Версия: **2026.07**  
Источник для UI: `src/data/publishRules.js` (модалка «Правила» в форме добавления звука). Этот файл — читаемая копия для документации.

---

## О платформе

RO.SMap — публичная аудиокарта полевых записей. Слушать можно без аккаунта; добавлять звуки, писать комментарии и вести экспедиции — после входа. Новые записи проходят **модерацию** перед появлением на карте.

## Правила добавления звука

| Код | Суть |
|-----|------|
| **S1** | Файл — корректный **WAV** |
| **S2** | Свой материал или права / лицензия |
| **S3** | Реальные координаты места записи |
| **S4** | Понятные название и описание |
| **S5** | Корректные UCS / CatID / теги |
| **S6** | Дата/время и техника по возможности |
| **S7** | Приемлемое качество, релевантность месту |
| **S8** | Корректная лицензия распространения |
| **S9** | Экспедиция = проект в UCS SourceID; платформенный ID только в метаданных |
| **S10** | Фото по теме; для звуковой прогулки — маршрут ≥ 2 точек |

## Модерация

- **Опубликовать** → статус «на модерации» + уведомление.
- **Одобрено** → на карте; подписчики могут узнать о новой записи.
- **Отклонено** → черновик + причина (часто код пункта, напр. `S3: …`). Клик по уведомлению открывает правку. Исправьте и отправьте снова (**M3** — не слать повтор без правок).

## Правила пользования

| Код | Суть |
|-----|------|
| **C1** | Без оскорблений, ненависти, травли |
| **C2** | Без спама и рекламы |
| **C3** | Без запрещённого / незаконного контента |
| **C4** | Уважительные комментарии |
| **C5** | Жалобы по делу |
| **C6** | Не передавать аккаунт / не выдавать себя за другого |

## Прослушивание и скачивание

Соблюдайте лицензию автора (**U1**), не снимайте требуемую атрибуцию из метаданных (**U2**), архивы экспедиций распространяйте с учётом лицензий файлов (**U3**).
