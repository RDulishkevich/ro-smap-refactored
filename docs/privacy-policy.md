# Политика конфиденциальности сервиса «Полёвка»

Версия: **2026.07**  
Источник для UI: `src/data/legalDocs.js`  
Сайт: https://www.polevka.art

## 1. Оператор

Оператор персональных данных сервиса «Полёвка» (аудиокарта полевых записей) — физическое лицо, применяющее специальный налоговый режим «Налог на профессиональный доход» (**самозанятый**): **Дулишкевич Роман Владимирович**.  
ИНН будет указан в этой политике после предоставления.  
Связь: чат поддержки в приложении; адрес — `support@polevka.art`. Документы (политика, соглашение, правила) доступны в приложении и для скачивания в PDF (печать → «Сохранить как PDF»).

## 2. Какие данные обрабатываем

- Логин, отображаемое имя, аватар, биография, ссылки и сведения об оборудовании.  
- Email (если указан) и факт его подтверждения.  
- Служебные коды, отправляемые на email (подтверждение адреса, восстановление пароля).  
- Сообщения, уведомления, комментарии, реакции, жалобы.  
- Метаданные и файлы звуковых записей, фото, координаты, UCS-классификация.  
- Анкета при регистрации (уровень умений, цели использования).  
- Согласие на обработку ПДн и время согласия.  
- Технические данные запросов к API (для безопасности и лимитов).

Сервис **не использует cookies** для сессии: токен и настройки хранятся в `localStorage` на устройстве пользователя.

## 3. Цели обработки

- Предоставление карты, кабинета, сообщений и модерации.  
- Публикация и каталогизация полевых записей.  
- Подтверждение email и восстановление доступа по email (коды через российский SMTP).  
- Связь по вопросам аккаунта.  
- Безопасность, предотвращение злоупотреблений, исполнение правил сервиса.  
- Исполнение требований законодательства РФ.

## 4. Правовые основания

- Согласие субъекта (регистрация, чекбокс согласия).  
- Исполнение пользовательского соглашения.  
- Законные интересы оператора (модерация, безопасность), если применимо.

## 5. Хранение и передача

Данные хранятся в облачном хранилище (включая приватный бакет для чувствительных сведений) на инфраструктуре, используемой сервисом, **на территории Российской Федерации** (объектное хранилище и API в Yandex Cloud).  
Третьи стороны-обработчики в объёме, необходимом для работы функций: **Яндекс** (карты, при необходимости перевод/облако); для отправки служебных писем (код подтверждения email, сброс пароля) — российский SMTP-провайдер (Reg.ru Mail / аналог в РФ), без использования зарубежных ESP для ПДн.  
Персональные данные не продаются. Трансграничная передача ПДн через зарубежные почтовые сервисы не применяется.

## 6. Срок хранения

Данные аккаунта и контента хранятся, пока аккаунт активен и контент опубликован, либо до удаления по запросу / решению модерации, если иное не требуется законом.

## 7. Права пользователя

Вы можете запросить уточнение, ограничение обработки, удаление аккаунта и связанных данных через **поддержку в приложении** или `support@polevka.art`.  
Отзыв согласия может ограничить использование функций, требующих аккаунта.  
Актуальные тексты политики и соглашения можно сохранить как PDF из модальных окон в приложении.

## 8. Изменения политики

Актуальная версия публикуется в приложении. Существенные изменения могут сопровождаться уведомлением.
